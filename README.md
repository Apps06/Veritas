# Veritas - Fake News Detector

A Chrome browser extension that detects fake news with AI-powered analysis. Get confidence scores, source credibility checks, and fact verification.

## Features

- **Fake News Detection** - Analyzes articles for potential misinformation
- **Source Credibility** - Checks if the source is reliable, questionable, or satire
- **Sensationalism Detection** - Identifies clickbait and sensational language
- **Fact Checking** - Integrates with Scira AI for fact verification
- **Credible Sources** - Provides links to fact-check websites
- **User Feedback** - Allows users to report accuracy of analyses

## Installation

1. Clone this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked**
5. Select the cloned folder

## Usage

1. Navigate to any news article
2. Click the Veritas extension icon
3. Click "Analyze Current Page"
4. Review the fake news risk score and details

## Configuration

Go to the extension options to configure your Scira AI API key for enhanced fact-checking capabilities.
